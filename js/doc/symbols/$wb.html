<static> $wb.Class(name, opts)
Function for creating simulated classes with support for multiple inheritance and calling overriden methods. Can be used stand alone - only dependency is $wb.Set 
var MySuperClass = $wb.Class("MySuperClass",{
     text:null,
     __construct:function(text) {
         this.text = text;
     }
     emit:function() {
         console.log(this.text);
     }
});
var MySubClass = $wb.Class("MySubClass",{
     __extends:[MySuperClass],
     __construct:function(text) {
         this.__super(text+ "world");
     }
});

var instance = new MySubClass("hello");
instance.emit(); // Emits hello world in the console.
Parameters:
{String} name
The name of the class. This will be the name of the constructor as well.
{Object} opts
The fields and methods for the class
{Class[]} opts.__extends Optional
Extend these classes. Notice that order matters and last wins in conflicts.
{Function} opts.__construct Optional
The constructor of the class
